Web / API Architecture — Colorful Flow

A layered view from user request to data, with security, caching, and delivery highlighted.
User & Client
[ USER / BROWSER / MOBILE APP ] Entry point
1. Browser or app sends request for https://example.com or API endpoint
2. CSP in browser ensures scripts execute safely
Edge Layer
CDN / Reverse Proxy / Security
  • Cloudflare / Akamai / AWS CloudFront: Receives request, checks WAF rules, blocks malicious payloads
  • Caches static content (images, CSS, JS) to reduce origin load
  • Caching at Edge: Varnish may serve cached HTML or cached API responses
  • Tag Manager: Google Tag Manager loads tracking scripts
  • Analytics: GA4 / Universal Analytics collects page visit & API usage data
  • Video Delivery: YouTube embeds delivered to browser
CDNWAFCachingAnalytics
Infrastructure Layer
Reverse Proxy / Load Balancing / API Gateway
  • App-Level Reverse Proxy / Load Balancer: Envoy routes requests to correct backend microservice (including API services)
  • Nginx / Apache / HAProxy distributes load across multiple app servers
  • API Gateway: Validates API keys, rate limits, and routes REST/GraphQL requests to appropriate microservice
  • Application Server: ColdFusion / Ruby / Node.js / Python processes requests; for API calls: parses request body (JSON), applies business logic
EnvoyNginxHAProxyAPI Gateway
Application Layer
Languages, Frameworks & APIs
  • Programming Languages: Ruby, CFML, Node.js
  • Frameworks / Libraries: jQuery, JSS
  • UI Framework: Bootstrap 4.6 for frontend layout
  • Static Site Generator: Jekyll 3.10.0 for static content
  • API Types:
    • REST API: /api/posts, /api/users
    • GraphQL API: Single endpoint /graphql
    • WebSocket API for real-time chat or notifications
  • API Security: Auth: OAuth 2.0, JWT; Validation & sanitization
  • Metadata Standards: Open Graph tags for social sharing
RESTGraphQLWebSocketOAuth2JWT
Hosting / PaaS
Static & Compute
  • GitHub Pages serves static content (if any)
  • AWS EC2 / Cloud Hosting runs application & API servers
GitHub PagesEC2Compute
Data Layer
Storage & External Services
  • Database: MySQL / PostgreSQL stores content, user data, API data
  • Object Storage: AWS S3 stores images, videos, static assets
  • External APIs:
    • Payment Gateway API (Stripe, PayPal)
    • Maps API (Google Maps)
    • AI API (image generation, NLP)
  • Internal API calls: Services talk to each other (microservice-to-microservice) via REST/gRPC
